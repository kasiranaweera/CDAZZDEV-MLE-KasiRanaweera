## Task 1 — Financial AI: LLM-Powered Equity Research Pipeline

An end-to-end equity research pipeline that fetches real-time OHLCV market data, computes five technical indicators from scratch (no TA-Lib), retrieves recent news headlines, performs per-headline LLM sentiment analysis via Groq, and generates a validated Buy/Hold/Sell trade signal — all presented as a styled HTML report.

**Highlights:**
- 3-level OHLCV fallback chain: yfinance → Stooq → Alpha Vantage
- RSI (Wilder), MACD, Bollinger Bands, SMA-50/200 computed from raw NumPy/Pandas
- Pydantic-validated structured LLM output for every API response
- Multi-panel matplotlib chart embedded in a styled HTML equity brief

📁 [`task1_financial/`](task1_financial/) · 📖 [Task 1 README](task1_financial/README.md)

## Task 2 — Generative AI: Mental Health Triage Fine-Tuning

A complete QLoRA fine-tuning pipeline that trains Phi-3-mini-4k-instruct on a 142-example mental health triage dataset generated by a Groq Llama-3-70B teacher model. Evaluation covers ROUGE-L, BERTScore F1, and a manual hallucination review on the held-out test set.

**Highlights:**
- Student: Phi-3-mini-4k-instruct (3.8B) · Teacher: Groq Llama-3-70B (different model — no circular signal)
- 4-bit NF4 QLoRA, LoRA rank=16, alpha=32, all 7 projection modules targeted
- ROUGE-L +0.0405 · BERTScore F1 +0.0189 · Hallucination rate 10%
- RAG fallback via ChromaDB + all-MiniLM-L6-v2 at confidence threshold 0.65

🤗 Fine-tuned model: [kasi-ranaweera/mental-health-triage-phi3-qlora](https://huggingface.co/kasi-ranaweera/mental-health-triage-phi3-qlora)

📁 [`task2_genai/`](task2_genai/) · 📖 [Task 2 README](task2_genai/README.md)

## Task 3 — Agentic Workflows: Multi-Agent Financial Research System

A production-grade two-agent financial research system built on LangGraph. Agent A (Data Analyst) fetches and analyses quantitative data; Agent B (Research Writer) synthesises qualitative context. Agents coordinate via a Pydantic-typed `DataBrief` contract and a critique/clarification loop, producing a full 3-section investment research report autonomously.

**Highlights:**
- 5 custom LangChain tools with schema-level tool restriction (`bind_tools()`)
- Pydantic `DataBrief` as the typed inter-agent handoff contract
- Persistent JSON cache, short-term session memory, and `agent_trace.jsonl` observability
- Streamlit dashboard + optional LangSmith tracing

📁 [`task3_agentic/`](task3_agentic/) · 📖 [Task 3 README](task3_agentic/README.md)
