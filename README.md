## Task 2 вЂ” Generative AI: Mental Health Triage Fine-Tuning

A complete QLoRA fine-tuning pipeline that trains Phi-3-mini-4k-instruct on a 142-example mental health triage dataset generated by a Groq Llama-3-70B teacher model. Evaluation covers ROUGE-L, BERTScore F1, and a manual hallucination review on the held-out test set.

**Highlights:**
- Student: Phi-3-mini-4k-instruct (3.8B) В· Teacher: Groq Llama-3-70B (different model вЂ” no circular signal)
- 4-bit NF4 QLoRA, LoRA rank=16, alpha=32, all 7 projection modules targeted
- ROUGE-L +0.0405 В· BERTScore F1 +0.0189 В· Hallucination rate 10%
- RAG fallback via ChromaDB + all-MiniLM-L6-v2 at confidence threshold 0.65

## Task 3 вЂ” Agentic Workflows: Multi-Agent Financial Research System

A production-grade two-agent financial research system built on LangGraph. Agent A (Data Analyst) fetches and analyses quantitative data; Agent B (Research Writer) synthesises qualitative context. Agents coordinate via a Pydantic-typed `DataBrief` contract and a critique/clarification loop, producing a full 3-section investment research report autonomously.

**Highlights:**
- 5 custom LangChain tools with schema-level tool restriction (`bind_tools()`)
- Pydantic `DataBrief` as the typed inter-agent handoff contract
- Persistent JSON cache, short-term session memory, and `agent_trace.jsonl` observability
- Streamlit dashboard + optional LangSmith tracing
